## 필터와 래퍼



#### 필터

필터(**filter**)란 글자그대로 여과기 역할을 하는 프로그램이다.

- 웹 브라우저가 웹 컴포넌트를 호출하면 필터가 대신 호출되고, 필터는 사전 작업을 수행한 다음에 웹 컴포넌트를 호출한다.

- 웹 컴포넌트가 할 일을 끝내면 실행의 제어는 다시 필터로 돌아가고,

웹 브라우저와 웹 컴포넌트 사이를 오가는 요청 메시지와 응답 메시지를 포장하는 프로그램이다.
