## 1.마크다운 기초

각종 마크다운 문법



- `#`은 대제목
- `##`은 중제목
- `-`은 동그라미
- `###`은 소제목
- `은 중요 단어 형광펜
  - 탭누르면 이렇게 빠짐
- 쉬프트 탭 누르면 다시 돌아감





## 2. 마크다운 문법



**코드하고 반대편에도 

**하면 강조함.

***세번 사이에 넣으면 볼드 + 옆으로 늬움



1.강조 : **이게강조된구문** -> **

2.기울임 : *이게 기울여진 구문* -> *

3.강조+기울임 : ***이게 강조+기울임***  -> ***



> 와 이거는 인용문을 쓸 떄 쓰면 되겠네 -> 
>
> 출력 방법 : >
