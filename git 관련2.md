### <새로 시작할 때>

git add .

git commit -m "name"

git push

안되면

git fetch origin BRANCH_NAME

git merge origin BRANCH_NAME

git pull origin BRANCH_NAME

다시

처음 컴퓨터 옮기면

=>

git config --global user.name 이름

git config --global user.email 이메일

git config --list

git remote add origin 레파지토리주소

git clone 레파지토리주소

git push -u origin master

끝!
